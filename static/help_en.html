help (en)

instructions

  1. click 'load file'
  2. select the .ptcop or .pttune you want played
  3. press play

features

views

  • unit: one row for each unit, showing note presses. higher volume/velocity = brighter
  • keyboard: piano roll. higher volume/velocity = brighter and more opaque

controls

  • play, pause, stop: starts/resumes, pauses, stops the piece
  • vol: controls master volume
  • zoom-x: controls measure width scale
  • zoom-y: controls key height scale in keyboard view
  • snap: controls what the playhead offset snaps to
  • 2x: doubles pixel zoom
  • dark: disables background in keyboard view

unit settings

  • left-click under : show/hide in keyboard view
  • left-click under : pin/unpin unit at top of both views
  • right-click under or when activated: change color
